# 5.1 类的定义与使用
- 类的定义
```Python
class Calculator:
#类的名称首字母大写
	price = 20
	name ='Good Calculator'
	def __init__(self,rlname,rlprice,H,width=6,weight=5):
	#=后赋值默认值
		self.n = rlname
		self.p = rlprice
		self.h = H
		self.wi = width
		self.we = weight
		#初始化类的属性

	def add(self,x,y):
	#定义类内函数
		res =x+y
		print(res)
```
- 对象的定义
```Python
c = Calculator('CC',20,30)
#实例化一个变量c，按顺序定义了形参rlname、rlprice、H
```
- 对象的调用
```Python
print('默认名字为',c.name)
#调用固有属性项

print('该实例c的名字为',c.n)
#调用实例

sum = c.add(3,4)
#使用类内定义的函数，结果为7
```
