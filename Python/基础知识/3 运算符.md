# 3.1 算术运算符
- 常用的有：加+、减-、乘`*`、除/、指数`**`、取余%、取整//
- py中的除/不同于C++，这里的除保留小数位，具体保留几位需手动控制
- 取整//相当于C++中的除/，结果向下取整
```Python
print(11//2)
# 结果为5.5去除小数位，即5
```
# 3.2 赋值运算符
- a += b 等价于 a = a + b
- 依此类推其他算术运算符的组合
# 3.3 逻辑运算符
- 常用的有：`==`、`>`、`<`、`>=`、`<=`
- 依据对条件的判断可以输出布尔值
```Python
result = 10 > 5
print(result)
# 结果为True

result = "666" == "667"
print(result)
# 结果为False

print(f"666{66 != 77}666")
# 结果为666True666
```
